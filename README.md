# 📒 Contact Saver

A simple and responsive Contact Saver web application built using **Node.js**, **MongoDB Atlas**, and **Mongoose**. Users can securely save, view, and delete contacts through a clean and user-friendly interface.

---

## 🚀 Features

* ➕ Add new contacts
* 📋 View all saved contacts
* 🗑️ Delete contacts
* 💾 Store data permanently using MongoDB Atlas
* 📱 Responsive and clean UI
* ⚡ Built without Express using Node.js, MongoDB and HTTP module

---

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/mohamedfahim18/Contact-Saver.git
```

### 2. Navigate to the project

```bash
cd Contact-Saver
```

### 3. Install dependencies

```bash
npm install
```

### 4. Create a `.env` file

```env
MONGO_URI=your_mongodb_connection_string
```

### 5. Start the server

```bash
node server.js
```

### 6. Open the application

```
http://localhost:3000
```
